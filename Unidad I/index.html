Conceptos Básicos

Computadora: Un Viaje a través de las Generaciones

Es fácil ver una computadora portátil o un smartphone hoy y pensar que siempre fueron así. Pero entender su evolución nos ayuda a apreciar la potencia que tenemos al alcance de la mano y a no temerle a la tecnología, porque vemos que es el resultado de un progreso constante y predecible.

Definición: Una computadora es, en esencia, una máquina electrónica que acepta datos (entrada), los procesa según un conjunto de instrucciones (programa) y produce un resultado (salida).

Evolución Histórica de los Sistemas de Computación :

1ª Generación (1940-1956): Válvulas de Vacío

Tecnología: Usaban tubos de vacío, enormes y frágiles, que generaban mucho calor.

Características: Eran gigantescas (ocupaban cuartos enteros), lentísimas y consumían una energía descomunal.

Ejemplo: ENIAC

2ª Generación (1956-1963): Transistores

Tecnología: Reemplazo de las válvulas por transistores.

Características: Más pequeñas, rápidas, eficientes y confiables.

3ª Generación (1964-1971): Circuitos Integrados

Tecnología: Múltiples transistores miniaturizados en un solo "chip" de silicio.

Características: Reducción drástica de tamaño y costo. Aumento masivo de la potencia.

4ª Generación (1971-Presente): Microprocesadores

Tecnología: Integración de toda la CPU en un solo chip.

Características: ¡Es la era en la que estamos! Esto permitió la creación de las computadoras personales (PC), las laptops y los smartphones.

5ª Generación (Presente y Futuro): Inteligencia Artificial y Computación Cuántica

Tecnología: Se centra en la paralelización masiva y el procesamiento inteligente.

Características: Sistemas con capacidad de aprendizaje (redes neuronales) y las primeras computadoras cuánticas.

ENIAC, el primer ordenador de propósito general.

Conclusión: En resumen, pasamos de cuartos llenos de tubos a tener supercomputadoras en nuestros bolsillos. Pregunta reflexiva: Si en 50 años pasamos de la ENIAC al iPhone, ¿cómo creen que será la computadora dentro de otros 50 años?

Hardware: El Cuerpo de la Computadora

Para entender cómo funciona una computadora, usemos una analogía: el Hardware es el cuerpo físico, todo lo que puedes tocar. El Software (que veremos luego) será la mente y el alma.

Definición: El Hardware es el conjunto de componentes físicos, tangibles, que conforman una computadora. Son las partes mecánicas, eléctricas y electrónicas.

Desglose de Componentes Principales

CPU / Procesador

Es el cerebro de la computadora. Ejecuta instrucciones de programas y realiza cálculos.

Memoria RAM

Memoria de trabajo temporal. Almacena datos e instrucciones de programas en ejecución.

Disco Duro

Memoria a largo plazo. Almacena sistema operativo, programas y archivos permanentemente.

Placa Base

Conecta y comunica todos los componentes entre sí. Es el sistema nervioso central.

Conclusión: El hardware funciona como un equipo: la placa base los conecta, la CPU procesa, la RAM mantiene los datos listos, y el disco duro los guarda para siempre. Dato curioso: La potencia de la CPU de un smartphone moderno es miles de veces superior a la computadora que guió el Apolo 11 a la Luna.

Software: La Mente y el Alma

De nada sirve el cuerpo (hardware) más avanzado sin una mente que lo dirija. El Software es ese conjunto de instrucciones intangibles que le dicen al hardware exactamente qué hacer.

Definición: El Software es el conjunto de programas, procedimientos, rutinas y datos asociados que permiten a la computadora realizar tareas específicas. Es la parte lógica, no física.

Clasificación del Software

Software de Sistema

Definición: Es el software fundamental que actúa como intermediario entre el usuario, el hardware y las demás aplicaciones.

Ejemplo: Sistemas Operativos (Windows, macOS, Linux)

Software de Aplicación

Definición: Son los programas que los usuarios ejecutan para realizar tareas específicas y productivas.

Ejemplo: Navegadores web, suites de oficina, editores de foto, videojuegos.

Software de Desarrollo

Definición: Son las herramientas que utilizan los programadores para crear nuevo software.

Ejemplo: Entornos de Desarrollo Integrados (IDE) como Visual Studio Code.

Conclusión: Hardware y Software son complementarios e inseparables. El hardware sin software es un cascarón vacío; el software sin hardware no tiene dónde existir. Pregunta reflexiva: ¿Podrían nombrar un ejemplo de Software de Aplicación que usen a diario y qué tarea les permite realizar?

Dispositivos de Almacenamiento: Del Cuarto al Bolsillo

Pensemos en la información personal más valiosa que tienen: fotos, documentos, trabajos de la universidad. ¿Dónde la guardan? Esa capacidad de almacenar datos de forma permanente es crucial.

Comparativa: Primera Generación vs. Actual

Primera Generación (Tarjetas Perforadas)

Medio: Agujeros en una tarjeta de cartón

Capacidad: 80 caracteres por tarjeta

Velocidad: Muy lenta

Fiabilidad: Baja

Generación Actual (Unidades SSD)

Medio: Chips de memoria flash

Capacidad: 256 GB a varios Terabytes

Velocidad: Extremadamente rápida

Fiabilidad: Muy alta

Disco Duro (Hard Drive)

Es el dispositivo de almacenamiento principal interno de la computadora, que utiliza uno o más platos o discos rígidos recubiertos de un material magnético para guardar datos de forma permanente.

Tipos principales:

HDD (Hard Disk Drive)

Tecnología: Discos magnéticos que giran a altas velocidades y cabezales mecánicos.

Ventaja: Bajo costo por gigabyte.

Desventaja: Más lento, sensible a golpes.

SSD (Solid State Drive)

Tecnología: Usa memoria flash (sin partes móviles).

Ventaja: Velocidad extremadamente superior, silencioso, resistente.

Desventaja: Costo más elevado por gigabyte.

Disco duro HDD vs SSD.

Memoria USB

Dispositivo de almacenamiento portátil que utiliza memoria flash y se conecta mediante un puerto USB. Su gran ventaja es la portabilidad y la facilidad de uso.

Tipos (clasificados por versión de USB):

  • USB 2.0: Velocidad de transferencia lenta (hasta 480 Mbps)
  • USB 3.0 / 3.1 / 3.2: Velocidad muy superior (desde 5 Gbps hasta 20 Gbps)
  • USB Type-C: Forma física del conector, reversible y moderno

Tipos de USB distinguidos por versiones.

Almacenamiento en la Nube

Es un modelo de almacenamiento donde los datos se guardan en servidores remotos a los que se accede a través de Internet. No se almacenan físicamente en tu dispositivo.

Tipos (por uso y acceso):

  • Nube Pública: Servidores de un proveedor compartidos por muchos usuarios (Google Drive, Dropbox)
  • Nube Privada: Infraestructura de uso exclusivo para una organización
  • Nube Híbrida: Combina nubes públicas y privadas

Tipos de almacenamiento en la nube.

Discos Externos

Básicamente son un HDD o SSD interno encerrado en una carcasa protectora con una conexión USB, diseñado para ser portátil y usado como respaldo o extensión del almacenamiento interno.

Un disco duro externo es un dispositivo de almacenamiento que se conecta a una computadora mediante puertos como USB, FireWire o Thunderbolt.

Conclusión: La nube es como alquilar una bodega de almacenamiento infinita a la que puedes acceder desde cualquier lugar del mundo con una conexión a Internet. Dato curioso: ¿Sabían que cuando publican una foto en Instagram o un video en TikTok, están usando almacenamiento en la nube?

Ambiente Windows: Nuestro Espacio de Trabajo Digital

Ahora que entendemos la máquina, es hora de aprender a movernos en ella. Windows es el Sistema Operativo que, como un buen anfitrión, nos da la bienvenida y nos proporciona un espacio para trabajar y jugar: el Escritorio.

Escritorio

Es la pantalla principal que vemos después de encender la computadora e iniciar sesión en Windows. Actúa como nuestra "mesa de trabajo" digital, donde colocamos los elementos que usamos con más frecuencia.

Escritorio de Window 11.

Barra de Tareas Windows 11

Es la barra larga horizontal que, por defecto, se encuentra en la parte inferior del escritorio. Es el centro de control de Windows.

Componentes y sus funciones (de izquierda a derecha):

  • 1 Widgets: Los widgets son elementos interactivos que muestran contenido dinámico y proporcionan acceso rápido a varias aplicaciones y funciones.
  • 2 Inicio:El menú Inicio es un centro central que proporciona acceso rápido a tus aplicaciones, configuración y archivos.
  • 3 Buscar: Windows Search es una potente función que le permite buscar archivos, aplicaciones, opciones de configuración e incluso resultados web directamente desde el escritorio.
  • 4 Vista de tareas: El botón vista de tareas de la barra de tareas está diseñado para mejorar la experiencia multitarea en Windows.
  • 5 Aplicaciones: El área de la barra de tareas usada para las aplicaciones de Windows está diseñada para proporcionar acceso rápido y administración de las aplicaciones.
  • 6 Bandeja del sistema: El área de la bandeja del sistema de la barra de tareas incluye varios botones y elementos que proporcionan acceso rápido a las funciones y notificaciones del sistema.

Configurar el diseño de la barra de tareas Windows 11.

Icono

Es una pequeña imagen gráfica que representa un objeto en la pantalla: un archivo, una carpeta, un programa o un acceso directo. Su función es permitirnos identificar y acceder a algo de manera visual e intuitiva.

Programas

También llamados aplicaciones o "apps", son software de aplicación que nos permiten realizar tareas específicas. Un programa es, por ejemplo, el navegador web con el que están viendo esta clase, el Word para escribir o el Paint para dibujar.

Conclusión: El Escritorio, con sus iconos y la Barra de Tareas, es nuestro punto de partida. Desde aquí controlamos todo lo que sucede en la computadora a través de los programas. Pregunta reflexiva: En su escritorio, ¿tienen iconos de archivos, de carpetas o de accesos directos? ¿Podrían identificar la diferencia?

El Teclado: Nuestro Instrumento de Comando

El teclado es nuestro principal medio para "hablar" con la computadora de forma precisa. Más allá de escribir textos, es una herramienta de control increíblemente poderosa.

Partes del Teclado

El teclado es un periférico de entrada que utiliza un conjunto de teclas o botones para actuar como palancas mecánicas o interruptores electrónicos, enviando instrucciones a la computadora.

Desglose de las Partes Principales:

  • Teclado Alfanumérico: La sección central con letras, números y símbolos.
  • Teclado Numérico (Keypad): Extremo derecho, agrupado como una calculadora.
  • Teclas de Función (F1 - F12): Fila superior, realizan acciones específicas.
  • Teclas de Control: Ctrl, Alt, Shift, Tecla de Windows.
  • Teclas de Navegación: Flechas de dirección, Inicio, Fin, Re Pág, Av Pág.
  • Teclas de Sistema: Imp Pant, Bloq Despl, Pausa/Inter.

Pregunta reflexiva para practicar: Si quisieran seleccionar todos los archivos de una carpeta y copiarlos a una memoria USB, ¿qué atajos de teclado usarían en el proceso? (La respuesta sería: Ctrl + A para seleccionar todo, luego Ctrl + C para copiar, ir a la USB y usar Ctrl + V para pegar).
